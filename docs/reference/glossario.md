# Glossario

Termini principali usati nell’applicazione e nella documentazione.

- **Commessa / Progetto**: contenitore principale; include uno o più preventivi.
- **Preventivo (Estimate)**: documento importato/gestito nel progetto. Può essere baseline “project”.
- **Baseline**: preventivo di progetto a cui si agganciano le offerte.
- **Offerta (Offer)**: ritorno di gara di una specifica impresa, in un round.
- **Round**: numero della tornata di offerte.
- **WBS**: struttura gerarchica; livelli 1–5 “spaziali”, 6–7 “commodity”.
- **Listino (Price List)**: insieme di voci prodotto/voce con prezzo (per-estimate).
- **Voci**: righe del computo (EstimateItem) o dell’offerta (OfferItem).
- **Aggregata (LX)**: offerta “per lista”, match per codice/descrizione (PriceListItem).
- **Dettagliata (MX)**: offerta “per computo”, match per progressivo/riga (EstimateItem).
- **Pending**: voce offerta senza match univoco; richiede scelta utente.
